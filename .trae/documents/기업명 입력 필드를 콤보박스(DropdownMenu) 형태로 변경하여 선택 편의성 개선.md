# 기업명 입력/선택 UI 개선 계획

사용자 피드백에 따르면 현재 `Autocomplete` 방식은 시각적으로 "선택 가능한 목록"이 있다는 것을 인지하기 어렵고, 빈 상태에서 목록이 바로 뜨지 않는 문제가 있습니다. 이를 해결하기 위해 입력과 선택이 모두 가능한 **콤보박스(DropdownMenu)** 형태로 UI를 변경하겠습니다.

## 변경 사항

### 1. `_CompanyNoteEditorSheet` (기업 노트 추가)
- **대상 파일**: [notes_screen.dart](file:///c:/Deadline%20Note/lib/src/screens/notes_screen.dart)
- **변경 내용**:
  - 기존 `RawAutocomplete`를 제거하고 Flutter의 `DropdownMenu<String>` 위젯을 적용합니다.
  - **드롭다운 버튼(▼)**이 명시적으로 표시되어 목록이 있음을 알 수 있게 합니다.
  - 필드를 누르거나 화살표를 누르면 **일정(캘린더) 및 현황에 등록된 회사 목록**이 즉시 표시됩니다.
  - 목록에 없는 회사는 기존처럼 **수기 입력**이 가능합니다.
  - 검색 필터링(`enableFilter: true`)을 활성화하여 타이핑 시 목록이 좁혀지도록 합니다.

### 2. `_InterviewSessionEditorSheet` (면접 회고 추가)
- **대상 파일**: [notes_screen.dart](file:///c:/Deadline%20Note/lib/src/screens/notes_screen.dart)
- **변경 내용**:
  - 마찬가지로 `RawAutocomplete`를 `DropdownMenu<String>`으로 교체합니다.
  - **[기존 기업 노트] + [일정/현황의 회사]**를 모두 합친 통합 목록을 제공합니다.
  - 목록에서 회사를 선택하면:
    - **기존 기업 노트에 있는 회사**인 경우: 시스템 내부 ID와 직무(Role)까지 자동으로 연동되어 채워집니다.
    - **일정에만 있는 회사**인 경우: 회사명만 자동으로 입력됩니다.
  - 수기 입력 시에도 자유롭게 새로운 회사명을 등록할 수 있습니다.

## 기대 효과
- 입력 필드 우측에 화살표 아이콘이 생겨 "선택 가능함"을 직관적으로 알 수 있습니다.
- "(주)리온전자"와 같이 이미 등록된 회사를 타이핑 없이 클릭 한 번으로 입력할 수 있습니다.
- 직무 정보 자동 연동 기능이 더 명확하게 동작합니다.
